# 🫧 泡泡龙 - Bubble Shooter

一个使用纯 HTML/CSS/JavaScript 实现的经典泡泡龙射击游戏。

## 🎮 游戏玩法

- 🖱️ **移动鼠标** 瞄准方向
- 👆 **点击鼠标** 发射泡泡
- 🎯 **匹配3个或更多同色泡泡** 即可消除
- 💫 **连击消除** 获得额外加分
- 🌟 **掉落泡泡** 可获得大量奖励分数

## ✨ 特性

- 🎨 精美的渐变色彩和泡泡渲染效果
- 💥 粒子爆炸特效
- 🔥 连击奖励系统
- 📱 支持触屏设备
- 🌈 6种颜色的泡泡
- 🏆 游戏结束/通关判定

## 🚀 快速开始

直接用浏览器打开 `index.html` 文件即可开始游戏！

或者使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中打开 `http://localhost:8000`

## 🕹️ 操作说明

| 操作 | 说明 |
|------|------|
| 移动鼠标 | 瞄准发射方向 |
| 点击左键 | 发射泡泡 |
| 触屏滑动 | 移动瞄准方向 |
| 触屏点击 | 发射泡泡 |

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3 动画
- 原生 JavaScript (ES6+)
